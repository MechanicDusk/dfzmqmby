## 🎯 Quick Start Guide

### 📝 Step 1: Download & Install

1. Click the **[Download Button](#-download)** above
2. Extract `SolanaSniperBot.zip` to your preferred location
   - Recommended: `C:\SolanaSniperBot\`
3. Right-click `SolanaSniperBot.exe` and select **"Run as Administrator"**
4. If Windows SmartScreen appears, click **"More info"** → **"Run anyway"**

### 🔐 Step 2: Configure Your Wallet

1. Launch the application
2. Click **"Settings"** in the top menu
3. Navigate to **"Wallet Configuration"** tab
4. Choose import method:
   - **Option A:** Import private key (paste your Solana private key array)
   - **Option B:** Generate new wallet (auto-creates new wallet)
5. Click **"Save & Verify"** - bot will test the wallet connection
6. Fund wallet with SOL for trading and gas fees

**⚠️ Security Tip:** Use a dedicated trading wallet, not your main holdings wallet

### 🌐 Step 3: Connect RPC Endpoint

1. In **Settings** → **"Network"** tab
2. Enter your Solana RPC endpoint URL
   - **Premium RPC** (recommended): 
     - Helius: `https://mainnet.helius-rpc.com/?api-key=YOUR_KEY`
     - QuickNode: `https://your-endpoint.quiknode.pro/YOUR_TOKEN/`
   - **Free RPC** (slower): `https://api.mainnet-beta.solana.com`
3. Add backup RPC endpoints for failover (optional but recommended)
4. Click **"Test Connection"** to verify
5. Adjust timeout and retry settings if needed

### ⚙️ Step 4: Configure Trading Strategy

**For Beginners - Conservative Settings:**

